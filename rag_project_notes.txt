Steps
-----
-> 1st, Create a backend API AI application
-> 2nd, connect nodejs app to the bakend api app to reflect response in event
-> 3rd, Run a local vectorstore using Docker container for storage
-> 4th, Run streamlit app integrated with inngest application




## Command to run main application ###
-> uv run uvicorn main:app



## Command to integrate inngest cli dev application with main app server ##
npx inngest-cli@latest dev -u http://127.0.0.1:8000/api/inngest --no-discovery

## npx => the nodejs package manager
## inngest-cli@latest dev -> Inngest dev server for response display
## http://127.0.0.1:8000/api/inngest -> main app backend host and route path
